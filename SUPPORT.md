# Getting help with macontrol

- **Questions & discussion** → [GitHub Discussions](https://github.com/amiwrpremium/macontrol/discussions)
- **Bugs** → open a [bug report](https://github.com/amiwrpremium/macontrol/issues/new?template=bug_report.yml)
- **Feature requests** → open a [feature request](https://github.com/amiwrpremium/macontrol/issues/new?template=feature_request.yml)
- **Security issues** → see [SECURITY.md](SECURITY.md) — please do **not** file a public issue

Before filing a bug, run `macontrol doctor` and paste the output. It
surfaces the macOS version, chip, capability gaps, missing brew deps, and
permission state — most issues are diagnosable from that alone.
